# NitikaNexus Chatbot

## Access the Application
The application is deployed and accessible via the following link: https://nitkanexus.vercel.app/

## Project Overview
NitikaNexus is an advanced chatbot application designed to allow users to better understand my skills, proficiencies, creativity, and more able myself.

## Tech Stack
Frontend: React.js (deployed on Vercel)
Backend: Python (Flask, deployed on Heroku)
API Integration: OpenAI API, LangChain
Deployment: Heroku (Backend), Vercel (Frontend)
